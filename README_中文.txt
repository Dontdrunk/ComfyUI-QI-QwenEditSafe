ComfyUI-QI-QwenEditSafe v1.5

新增：**QI • 文生图编辑（中文·一致性增强）**
- 中文参数名，更贴近中文用户习惯
- “调色板锚定”：从输入图聚类 K 个主色块，作为额外图像 token 喂给 Qwen 的图文编码器，
  直接在**推理阶段**约束色调/风格，减少“冷→暖”“色感漂移”
- 仍支持 reference_latents / reference_pixels 注入；兼容 Nunchaku/FLUX（自动网格=64）

推荐接线：
  UNET/CLIP/VAE 加载 → **QI_中文节点**.条件 → KSampler.positive
                      → **QI_中文节点**.潜空间 → KSampler.latent_image
  KSampler.LATENT → VAEDecode → SaveImage

中文参数速览：
- 不缩放_保持原尺寸：尽量不缩放，最大保真（显存不足可关）
- 目标总像素：需要缩放时的目标像素（如 1MP）
- 网格倍数 / 对齐策略：对齐到 8/16（标准）或 64（FLUX）倍数；“就近”更锐，“向下”更稳
- 注入模式：自动/仅latent/仅像素/两者都注入（FLUX 建议“仅像素”或“两者都注入”）
- 辅助原图参与编码：把轻下采样原图也喂给编码器（非 FLUX 场景可开）
- 调色板锚定 / 颜色数 / 重复次数：越大越“锁色”，但可能略牺牲自由度
- 高频轻加强：在进入编码前轻锐化一点点，保纹理，不是后期滤镜

预设：
- 标准 Qwen：不缩放=True，网格=8，就近对齐，注入=两者都注入，调色板=开(K=5, 重复=2)
- FLUX/Nunchaku：不缩放=True，网格=64，向下对齐，注入=仅像素，调色板=关（如兼容性良好可试开）
